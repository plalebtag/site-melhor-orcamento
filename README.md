# Melhor Orçamento — Site

Landing page do Melhor Orçamento — SaaS B2B que compara orçamentos de fornecedores
e entrega a lista de compra de menor custo por fornecedor.

## Estrutura

- `highfi/index.html` — versão **atual** com identidade visual da marca aplicada
- `lowfi/index.html` — wireframe original em P&B (referência)
- `assets/` — logo e símbolo da marca (SVG)
- `index.html` — redireciona para a versão high-fi

## Rodar localmente

Abra `highfi/index.html` (ou `lowfi/index.html`) no navegador.
