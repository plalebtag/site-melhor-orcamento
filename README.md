# Melhor Orçamento — Site

Wireframe em baixa fidelidade (P&B) da landing page do Melhor Orçamento — SaaS B2B
que compara orçamentos de fornecedores e entrega a lista de compra de menor custo
por fornecedor.

- `lowfi/index.html` — wireframe atual (HTML/CSS puro, sem dependências)
- `index.html` — redireciona para o wireframe

## Rodar localmente

Abra `lowfi/index.html` no navegador.
